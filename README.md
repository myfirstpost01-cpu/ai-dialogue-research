# ai-dialogue-research
Исследовательский путь: от первого диалога с ИИ к созданию собственных инструментов управления. В процессе работы я систематизировала сбои, ограничения и противоречия в поведении моделей. Это привело к разработке режимов управления, протоколов проверки данных и модульных архитектур. Проект — открытая документация этого опыта.
